Данный плагин выводит партнерский виджет покупки книг от ЛитРес.

== Description ==

Данный плагин с помощью шорткода <strong>[litres]</strong> выводит партнерский виджет покупки книг от [ЛитРес](https://www.litres.ru/partners/). 

После установки плагина не забудьте проверить его настройки в &#171;<strong>Настройках\LitRes Widget</strong>&#187; (необходимо установить партнерский ID).

Полный шорткод плагина:

`[litres author='Автор книги' title='Название книги']` 

Для вставки шорткода плагина вы можете воспользоваться кнопкой плагина в визуальном редакторе или вставить его вручную.

Если вам понравился этот плагин, то, <strong>пожалуйста</strong>, поставьте ему 5 звезд.


== Frequently Asked Questions ==

= Инструкция по установке =

1. Установите плагин через меню &#171;<strong>Плагины\Добавить новый</strong>&#187; (искать &#171;<strong>LitRes Widget</strong>&#187;).
2. Активируйте плагин в меню &#171;<strong>Плагины</strong>&#187;.
3. Зайдите в настройки плагина &#171;<strong>Настройки\LitRes Widget</strong>&#187; и сохраните их.
4. Это все.

= Где взять партнерский ID? =

Чтобы его получить вам необходимо подключиться к партнерской программе [ЛитРес](https://www.litres.ru/partners/). 

= Виджет выводится криво, что делать? =

ЛитРес-виджет выводит html-таблицу, а практически каждая тема для WordPress имеет свое собственное оформление для html-таблиц, которое и наследуется ЛитРес-виджетом. В плагине исправлены основные &#171;косяки&#187; такого наследования, но проверить работу на каждой существующей теме невозможно. В случае проблем с css-оформлением виджета на вашей теме пишите на [форуме](https://wordpress.org/support/plugin/litres-widget) - я вам обязательно помогу.
